协方差（Covariance）：衡量两个变量如何一起变化

2026-05-17 14:24 33 min read -- 人阅读

协方差到底是什么：看懂两个变量如何一起变化

引言：方差看自己，协方差看彼此

在前面学习方差时，我们关心的是：

一个变量自身波动得有多厉害？

比如一个班级的成绩是否稳定，一个基金的收益是否波动，一个生产线的产品尺寸是否一致。

但现实中，我们经常不只关心一个变量，而是关心两个变量之间有没有关系。

例如：

  • 学习时间越长，考试成绩是否越高？
  • 广告投入越多，销售额是否越高？
  • 气温升高，冰淇淋销量是否增加？
  • 房屋面积越大，房价是否越高？
  • 股票 A 上涨时，股票 B 是否也容易上涨？

这些问题都不是在问单个变量的波动，而是在问：

两个变量是否会一起变化？

这就是协方差要解决的问题。

一句话理解：

方差衡量一个变量自己的波动，协方差衡量两个变量共同变化的方向。

第一部分：协方差的直觉

1. 什么是协方差？

协方差的英文是 Covariance。

拆开看：

  • co 表示共同。
  • variance 表示方差、波动。

所以协方差可以理解为：

两个变量共同波动的程度。

更具体一点：

当 X 高于自己的平均值时，Y 是否也倾向于高于自己的平均值？当 X 低于自己的平均值时，Y 是否也倾向于低于自己的平均值？

如果答案是“是”，协方差通常为正。

如果答案是“相反”，协方差通常为负。

如果两者没有明显同步关系，协方差接近 0。

2. 三种关系：正、负、接近 0

2.1 正协方差

如果 X 增大时，Y 也倾向于增大；X 减小时，Y 也倾向于减小，那么协方差为正。

例如：

学习时间越长，考试成绩越高
房屋面积越大，房屋价格越高
广告投入越多，销售额越高

这类关系的特点是：

两个变量同向变化。

2.2 负协方差

如果 X 增大时，Y 倾向于减小；X 减小时，Y 倾向于增大，那么协方差为负。

例如：

商品价格越高，购买数量可能越少
运动时间越长，体脂率可能越低
距离市中心越远，房价可能越低

这类关系的特点是：

两个变量反向变化。

2.3 协方差接近 0

如果 X 和 Y 没有明显的线性同步关系，协方差接近 0。

例如：

鞋码和数学成绩
手机电量和今天午饭吃什么
身份证尾号和收入水平

这类关系的特点是：

两个变量没有明显的线性共同变化趋势。

注意：协方差接近 0 不一定表示两个变量完全没有关系，只能说明它们没有明显的线性共同变化关系。

第二部分：协方差公式

3. 从方差公式到协方差公式

方差的公式是：

Var(X) = E[(X - E(X))²]

它衡量的是 X 自己围绕均值的波动。

如果把平方展开成“自己乘自己”：

Var(X) = E[(X - E(X))(X - E(X))]

这时我们把其中一个 X 换成 Y，就得到协方差：

Cov(X, Y) = E[(X - E(X))(Y - E(Y))]

这就是协方差的核心公式。

它的意思是：

看 X 相对均值的偏差，与 Y 相对均值的偏差，是否经常同号或异号。

如果两个偏差经常同号，乘积为正，协方差为正。

如果两个偏差经常异号，乘积为负，协方差为负。

4. 总体协方差

如果我们掌握了全部数据，使用总体协方差：

Cov(X, Y) = Σ[(xᵢ - μₓ)(yᵢ - μᵧ)] / n

其中：

  • xᵢ 是 X 的第 i 个观测值。
  • yᵢ 是 Y 的第 i 个观测值。
  • μₓ 是 X 的总体均值。
  • μᵧ 是 Y 的总体均值。
  • n 是数据数量。

5. 样本协方差

如果我们只有样本数据，用样本协方差：

Cov(X, Y) = Σ[(xᵢ - x̄)(yᵢ - ȳ)] / (n - 1)

其中：

  • x̄ 是 X 的样本均值。
  • ȳ 是 Y 的样本均值。
  • n - 1 是自由度修正。

和样本方差一样，样本协方差通常除以 n - 1，用于修正样本估计总体时的偏差。

第三部分：一步步计算协方差

6. 例子：学习时间和考试成绩

假设我们记录了 5 个学生的学习时间和考试成绩：

学生 学习时间 X（小时） 考试成绩 Y（分）
A 1 50
B 2 60
C 3 70
D 4 80
E 5 90

6.1 计算均值

学习时间均值：

x̄ = (1 + 2 + 3 + 4 + 5) / 5 = 3

成绩均值：

ȳ = (50 + 60 + 70 + 80 + 90) / 5 = 70

6.2 计算偏差

学生 xᵢ - x̄ yᵢ - ȳ
A -2 -20
B -1 -10
C 0 0
D 1 10
E 2 20

6.3 偏差相乘

学生 (xᵢ - x̄)(yᵢ - ȳ)
A 40
B 10
C 0
D 10
E 40

偏差乘积之和：

40 + 10 + 0 + 10 + 40 = 100

6.4 计算总体协方差

如果这 5 个学生就是全部对象：

Cov(X, Y) = 100 / 5 = 20

协方差为正，说明学习时间和考试成绩同向变化。

6.5 计算样本协方差

如果这 5 个学生只是从更大群体里抽取的样本：

Cov(X, Y) = 100 / (5 - 1) = 25

7. 反向变化的例子

假设商品价格和销量如下：

商品 价格 X 销量 Y
A 10 100
B 20 80
C 30 60
D 40 40
E 50 20

价格越高，销量越低。

这时：

  • 价格高于均值时，销量低于均值。
  • 价格低于均值时，销量高于均值。

两个偏差经常异号，偏差乘积多为负，所以协方差为负。

第四部分：如何解释协方差？

8. 协方差的符号比大小更重要

协方差最直观的信息是符号：

协方差 含义 关系
大于 0 两个变量同向变化 X 高于均值时，Y 也倾向于高于均值
小于 0 两个变量反向变化 X 高于均值时，Y 倾向于低于均值
接近 0 没有明显线性共同变化 不代表完全无关系

但是协方差的数值大小不太好直接解释。

原因是：

协方差受变量单位影响。

例如：

  • 身高用“米”表示，协方差可能很小。
  • 身高用“厘米”表示，协方差会变大很多。

这导致协方差不能直接用来比较不同变量对之间的相关强弱。

所以，协方差更适合判断方向，而不是直接判断强度。

9. 协方差为什么难解释？

假设我们研究身高和体重：

身高单位：厘米
体重单位：千克

协方差的单位就是：

厘米 × 千克

这个单位很奇怪。

如果把身高从厘米改成米，协方差数值会缩小 100 倍。

但两个变量之间的关系并没有变。

这说明：

协方差的绝对值依赖量纲，不适合单独拿来解释相关强度。

为了解决这个问题，统计学引入了相关系数。

第五部分：协方差和相关系数

10. 相关系数是标准化后的协方差

皮尔逊相关系数的公式是：

r = Cov(X, Y) / (σₓ σᵧ)

其中：

  • Cov(X, Y) 是 X 和 Y 的协方差。
  • σₓ 是 X 的标准差。
  • σᵧ 是 Y 的标准差。

这相当于把协方差除以两个变量各自的波动尺度。

结果是：

-1 ≤ r ≤ 1

相关系数的解释就清楚很多：

r 的范围 含义
接近 1 强正线性相关
接近 -1 强负线性相关
接近 0 弱线性相关或无线性相关

11. 协方差和相关系数的区别

指标 是否受单位影响 取值范围 主要用途
协方差 受单位影响 无固定范围 判断共同变化方向
相关系数 不受单位影响 [-1, 1] 判断线性关系方向和强度

简单理解：

  • 协方差告诉你两个变量是同向还是反向变化。
  • 相关系数告诉你这种线性关系有多强。

所以实际分析中，经常先讲协方差的思想，再使用相关系数做解释。

12. 相关不等于因果

无论是协方差还是相关系数，都只能描述变量之间的统计关系。

它们不能直接证明因果。

例如：

冰淇淋销量和溺水人数可能正相关

但不能说“吃冰淇淋导致溺水”。

更合理的解释是：

夏天气温升高，冰淇淋销量增加，同时游泳人数增加，溺水风险也增加。

这里真正的共同因素可能是气温。

所以看到协方差或相关系数时，一定要记住：

相关关系是线索，不是结论。

第六部分：协方差矩阵

13. 从两个变量到多个变量

如果只有两个变量，我们只需要一个协方差：

Cov(X, Y)

但在机器学习中，我们经常有很多特征：

身高、体重、年龄、收入、消费金额、点击次数、停留时长...

这时我们希望同时知道：

  • 每个变量自己的方差。
  • 每两个变量之间的协方差。

于是就有了协方差矩阵。

14. 协方差矩阵长什么样？

假设有三个变量 X₁、X₂、X₃，协方差矩阵是：

        X₁          X₂          X₃
X₁  Var(X₁)   Cov(X₁,X₂)  Cov(X₁,X₃)
X₂  Cov(X₂,X₁) Var(X₂)    Cov(X₂,X₃)
X₃  Cov(X₃,X₁) Cov(X₃,X₂) Var(X₃)

它有两个重要特点：

  1. 对角线是每个变量自己的方差。
  2. 非对角线是不同变量之间的协方差。

因为：

Cov(X, Y) = Cov(Y, X)

所以协方差矩阵是对称矩阵。

15. 协方差矩阵有什么用？

协方差矩阵是很多机器学习和数据分析方法的基础。

典型应用包括：

  • PCA 主成分分析：寻找数据方差最大的方向，用于降维和压缩。
  • 多元高斯分布：描述多个变量之间的联合分布。
  • 金融资产组合：分析不同资产收益率之间的共同波动。
  • 特征冗余分析：判断多个特征是否携带重复信息。
  • 异常检测：识别不符合整体协方差结构的数据点。

尤其是 PCA，它和协方差关系非常紧密。

PCA 要找的是：

数据在哪些方向上变化最大？

而“变化最大”这个问题，正是通过协方差矩阵来表达的。

第七部分：协方差在机器学习中的意义

16. 特征关系分析

机器学习建模前，我们常常要分析特征之间的关系。

如果两个特征高度相关，说明它们可能携带相似信息。

例如预测房价时：

房屋面积
卧室数量
客厅面积
总价

房屋面积和客厅面积可能高度相关。

如果特征之间高度冗余，可能会影响模型解释性，也可能带来多重共线性问题。

协方差和相关系数可以帮助我们识别这些关系。

17. PCA 降维

PCA 的基本思想是：

找到数据变化最大的方向，把数据投影到这些方向上。

这些方向来自协方差矩阵的特征向量。

协方差矩阵告诉我们数据在不同方向上的联合变化情况。

特征值告诉我们某个方向上保留了多少方差信息。

所以 PCA 本质上是在做：

用尽可能少的维度，保留尽可能多的数据波动信息。

18. 金融风险建模

在投资组合中，我们不仅关心每个资产自己的波动，还关心资产之间是否一起涨跌。

如果两个资产协方差很高：

它们可能经常一起涨、一起跌。

如果两个资产协方差较低或为负：

它们可能有一定分散风险的作用。

所以资产组合优化中，协方差矩阵是非常核心的输入。

第八部分：用代码计算协方差

19. Python 手写协方差

x = [1, 2, 3, 4, 5]
y = [50, 60, 70, 80, 90]

x_mean = sum(x) / len(x)
y_mean = sum(y) / len(y)

cov_population = sum(
    (xi - x_mean) * (yi - y_mean)
    for xi, yi in zip(x, y)
) / len(x)

cov_sample = sum(
    (xi - x_mean) * (yi - y_mean)
    for xi, yi in zip(x, y)
) / (len(x) - 1)

print(cov_population)  # 20.0
print(cov_sample)      # 25.0

20. NumPy 计算协方差

import numpy as np

x = np.array([1, 2, 3, 4, 5])
y = np.array([50, 60, 70, 80, 90])

print(np.cov(x, y))

输出的协方差矩阵大致是：

[[  2.5  25. ]
 [ 25.  250. ]]

含义是：

  • 2.5 是 X 的样本方差。
  • 250 是 Y 的样本方差。
  • 25 是 X 和 Y 的样本协方差。

NumPy 的 np.cov 默认计算样本协方差，也就是分母使用 n - 1。

21. Pandas 计算协方差和相关系数

import pandas as pd

df = pd.DataFrame({
    "study_hours": [1, 2, 3, 4, 5],
    "score": [50, 60, 70, 80, 90]
})

print(df.cov())
print(df.corr())

df.cov() 计算协方差矩阵。

df.corr() 计算相关系数矩阵。

实际数据分析中，通常会同时看这两个结果：

  • 协方差用于理解共同波动。
  • 相关系数用于比较关系强弱。

第九部分：常见误区

22. 误区一：协方差为 0 就一定没有关系

不一定。

协方差主要衡量的是线性共同变化。

如果两个变量之间存在非线性关系，协方差可能仍然接近 0。

比如：

Y = X²

当 X 在正负对称区间取值时，X 和 Y 的协方差可能接近 0，但它们显然存在关系。

所以协方差为 0，只能说明：

没有明显线性共同变化，不代表没有任何关系。

23. 误区二：协方差越大，关系一定越强

不一定。

协方差受单位影响。

如果把身高单位从米改成厘米，协方差数值会放大 100 倍。

但变量之间的关系并没有变强。

所以比较关系强弱时，更适合使用相关系数。

24. 误区三：协方差能证明因果

不能。

协方差只能说明两个变量是否有共同变化趋势，不能证明谁导致了谁。

比如广告投入和销售额正协方差，可能是广告提高了销售，也可能是旺季时企业同时增加广告预算和销售自然增长。

要证明因果，需要实验设计、随机对照、因果推断等更严格的方法。

第十部分：核心总结

25. 一句话记住协方差

协方差衡量两个变量是否倾向于一起变化。

几个关键结论：

  1. 方差衡量一个变量自己的波动。
  2. 协方差衡量两个变量共同变化的方向。
  3. 协方差大于 0，表示两个变量同向变化。
  4. 协方差小于 0，表示两个变量反向变化。
  5. 协方差接近 0，表示没有明显线性共同变化。
  6. 协方差受单位影响，因此数值大小不容易直接解释。
  7. 相关系数是标准化后的协方差，更适合比较线性关系强弱。
  8. 协方差矩阵是 PCA、金融风险建模、多元统计分析的重要基础。

如果方差回答的是：

一个变量自己波动得有多厉害？

那么协方差回答的是：

两个变量是不是在一起波动？

学会协方差，就等于从“看单个变量”进入了“看变量关系”的世界。
