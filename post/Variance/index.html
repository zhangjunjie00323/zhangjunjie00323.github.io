方差（Variance）：衡量数据波动的核心尺度

2026-05-17 10:10 30 min read -- 人阅读

方差到底是什么：从平均数到数据波动的完整理解

引言：平均数不够用

如果你只看平均数，很多数据会显得“差不多”。

比如有两组考试成绩：

A 组：80, 80, 80, 80, 80
B 组：60, 70, 80, 90, 100

它们的平均分都是 80。

但直觉告诉我们，这两组数据完全不一样：A 组非常稳定，每个人都刚好 80；B 组波动很大，有人 60，有人 100。

这就说明：平均数只能告诉我们数据集中在哪里，却不能告诉我们数据分散得有多厉害。

而方差要解决的问题正是：

数据整体上离平均值有多远？

方差不是在看“中心”，而是在看“波动”。它是统计学、概率论、机器学习、金融风险、质量控制中极其基础的概念。

第一部分：方差的直觉

1. 什么是方差？

一句话理解：

方差是每个数据点到平均值距离的平方的平均值。

这句话里有三个关键词：

  • 平均值：先找到数据的中心。
  • 距离：看每个数据离中心有多远。
  • 平方：把距离平方后再取平均。

假设一组数据是：

2, 4, 6

它们的平均值是：

(2 + 4 + 6) / 3 = 4

每个数据到平均值的偏差是：

2 - 4 = -2
4 - 4 = 0
6 - 4 = 2

这些偏差说明：

  • 2 比平均值小 2。
  • 4 刚好等于平均值。
  • 6 比平均值大 2。

如果只看偏差本身，你会发现：

-2 + 0 + 2 = 0

正负抵消了。

所以我们不能直接把偏差加起来求平均，否则任何一组数据相对于平均值的偏差和都会等于 0。

这就是方差要引入“平方”的原因之一。

2. 为什么要平方？

方差的核心操作是：

偏差 -> 平方 -> 求平均

也就是：

数据点与平均值的差距越大，平方后惩罚越重。

比如偏差是 1、2、5：

1² = 1
2² = 4
5² = 25

偏差从 2 增加到 5，只增加了 2.5 倍；但平方后从 4 增加到 25，增加了 6.25 倍。

这意味着方差非常重视离群值。一个特别远的数据点，会显著拉大方差。

平方有三个作用：

  1. 避免正负抵消：负偏差平方后变成正数。
  2. 放大较大偏差：越远的数据点影响越大。
  3. 带来良好的数学性质：平方函数连续、可导、容易优化，是统计建模和机器学习中非常重要的原因。

第二部分：方差公式

3. 总体方差

如果我们掌握了研究对象的全部数据，这时计算的是总体方差。

设总体数据为：

x₁, x₂, x₃, ..., xₙ

总体平均值为：

μ = (x₁ + x₂ + x₃ + ... + xₙ) / n

总体方差为：

σ² = [(x₁ - μ)² + (x₂ - μ)² + ... + (xₙ - μ)²] / n

也可以写成：

σ² = Σ(xᵢ - μ)² / n

其中：

  • σ² 表示总体方差。
  • μ 表示总体均值。
  • xᵢ 表示第 i 个数据。
  • n 表示数据总数。

总体方差适用于：你手上已经有全部数据，不需要通过样本推断总体。

4. 样本方差

现实中，我们常常拿不到全部数据，只能拿到一部分样本。

比如：

  • 想知道全国学生的身高波动，不可能测量每一个学生。
  • 想知道某个工厂产品重量是否稳定，只能抽检一部分产品。
  • 想知道用户消费金额的波动，只能基于一段时间内的样本。

这时我们计算的是样本方差。

样本均值为：

x̄ = (x₁ + x₂ + x₃ + ... + xₙ) / n

样本方差为：

s² = [(x₁ - x̄)² + (x₂ - x̄)² + ... + (xₙ - x̄)²] / (n - 1)

也可以写成：

s² = Σ(xᵢ - x̄)² / (n - 1)

这里最容易让人疑惑的是：

为什么样本方差除以 n - 1，而不是 n？

5. 为什么样本方差要除以 n - 1？

先说结论：

样本方差除以 n - 1，是为了修正用样本估计总体方差时的系统性低估。

原因在于，样本均值 x̄ 是用这组样本自己算出来的。它天然会比真实总体均值更贴近这组样本。

也就是说，当我们计算：

xᵢ - x̄

时，每个样本点到样本均值的距离，通常会比它到真实总体均值的距离更小一些。

这样直接除以 n，会让方差偏小。

所以统计学用 n - 1 来修正这个偏差。这个修正叫作 贝塞尔校正（Bessel's correction）。

更直观地说：

如果你知道一组数据的平均值，那么在 n 个数据里，只有 n - 1 个可以自由变化。最后一个数据会被平均值约束住。

例如 3 个数的平均值是 10：

x₁ + x₂ + x₃ = 30

如果你已经知道了 x₁ 和 x₂，那么 x₃ 就被确定了：

x₃ = 30 - x₁ - x₂

所以样本方差真正拥有的自由信息量是 n - 1，这也叫 自由度。

第三部分：一步步计算方差

6. 例子：计算一组数据的方差

假设有一组数据：

2, 4, 6, 8

6.1 计算平均值

x̄ = (2 + 4 + 6 + 8) / 4 = 5

6.2 计算偏差

2 - 5 = -3
4 - 5 = -1
6 - 5 = 1
8 - 5 = 3

6.3 计算偏差平方

(-3)² = 9
(-1)² = 1
1² = 1
3² = 9

6.4 求偏差平方和

9 + 1 + 1 + 9 = 20

6.5 计算总体方差

如果这 4 个数就是全部数据：

σ² = 20 / 4 = 5

6.6 计算样本方差

如果这 4 个数只是样本：

s² = 20 / (4 - 1) = 20 / 3 ≈ 6.67

同一组数据，作为总体或样本时，方差公式不同。

7. 方差越大说明什么？

方差越大，说明数据越分散。

方差越小，说明数据越集中。

比如：

A 组：49, 50, 51
B 组：10, 50, 90

两组平均值都是 50。

但 A 组大多数数据都贴近 50，波动很小；B 组离 50 很远，波动很大。

所以：

  • A 组方差小。
  • B 组方差大。

方差让我们看到平均数看不到的信息。

第四部分：方差、标准差与单位问题

8. 方差的单位为什么有点奇怪？

方差有一个不太直观的地方：

方差的单位是原始单位的平方。

如果数据单位是“分”，方差单位就是“平方分”。

如果数据单位是“元”，方差单位就是“平方元”。

这显然不好解释。

比如考试成绩的方差是 25，你可以勉强理解为波动程度；但如果说“25 平方分”，就不自然了。

这就是为什么我们经常使用标准差。

9. 标准差是什么？

标准差就是方差的平方根。

总体标准差：

σ = √σ²

样本标准差：

s = √s²

标准差的好处是：

标准差和原始数据拥有相同单位。

如果成绩的方差是 25，那么标准差是：

√25 = 5

这就可以解释为：

数据通常在平均值附近上下波动约 5 分。

严格来说，标准差不是“平均偏差”，但在直觉上，它确实可以帮助我们理解数据围绕均值波动的典型尺度。

10. 方差和标准差的区别

指标 公式 单位 适合做什么
方差 偏差平方的平均 原单位的平方 数学推导、模型优化、理论分析
标准差 方差的平方根 原单位 直观解释数据波动

简单理解：

  • 方差更适合计算。
  • 标准差更适合解释。

在机器学习中，很多损失函数和统计模型都直接使用平方误差或方差思想，因为平方形式便于求导和优化。

在业务分析中，人们更喜欢标准差，因为它和原始数据单位一致，更容易沟通。

第五部分：方差的另一种计算公式

11. 方差的快捷公式

总体方差还有一个常见等价公式：

σ² = E(X²) - [E(X)]²

翻译成人话就是：

方差 = 平方的平均 - 平均的平方

对于一组总体数据：

σ² = (x₁² + x₂² + ... + xₙ²) / n - μ²

举个例子：

数据：2, 4, 6
平均值：μ = 4
平方平均：(2² + 4² + 6²) / 3 = (4 + 16 + 36) / 3 = 56 / 3
平均的平方：4² = 16
方差：56 / 3 - 16 = 8 / 3

这个公式在程序计算、概率论推导和机器学习中非常常见。

不过在初学时，更建议先理解：

方差 = 到均值距离平方的平均

因为它更符合直觉。

第六部分：方差在现实中的应用

12. 学生成绩：看稳定性

两个班级平均分相同，不代表教学效果一样。

A 班：成绩集中在 75 到 85 之间
B 班：成绩从 40 到 100 都有

两者平均分可能都差不多，但 B 班方差更大，说明学生差异更明显。老师可能需要分层教学，而不是只看平均分。

13. 金融投资：看风险

在金融中，收益率的方差常常用来衡量风险。

如果两个投资产品平均收益率都为 8%：

  • 产品 A 每年收益稳定在 7% 到 9%。
  • 产品 B 可能一年赚 30%，也可能一年亏 20%。

产品 B 的方差更大，风险也更高。

这就是为什么投资里不能只看收益，还必须看波动。

14. 工业生产：看质量稳定性

假设一个零件标准重量是 100 克。

两个生产线平均重量都接近 100 克：

  • 生产线 A：99.8、100.1、100.0、99.9
  • 生产线 B：96、103、98、104

平均值可能相近，但生产线 B 的方差更大，说明产品质量不稳定。

在质量控制中，方差比平均值更能暴露生产过程是否可靠。

15. 机器学习：看误差和信息量

机器学习中，方差的思想无处不在。

比如均方误差（MSE）：

MSE = Σ(yᵢ - ŷᵢ)² / n

它本质上就是预测值与真实值之间偏差平方的平均。

此外，机器学习还经常讨论：

  • 偏差（Bias）：模型预测的系统性误差。
  • 方差（Variance）：模型对训练数据变化的敏感程度。

一个高方差模型，可能在训练集上表现很好，但换一批数据就不稳定，这通常意味着过拟合。

第七部分：容易混淆的概念

16. 方差和平均绝对偏差

除了方差，还有一种衡量波动的方法叫 平均绝对偏差：

平均绝对偏差 = |x₁ - x̄| + |x₂ - x̄| + ... + |xₙ - x̄| 的平均

它不平方，而是取绝对值。

两者区别：

指标 对大偏差的态度 数学性质 直观程度
方差 放大大偏差 更适合推导和优化 稍弱
平均绝对偏差 线性对待偏差 某些场景不如平方方便 更直观

方差不是唯一衡量离散程度的方式，但它是数学性质最重要、应用最广泛的一种。

17. 方差和协方差

方差衡量的是：

一个变量自身的波动程度。

协方差衡量的是：

两个变量一起变化的方向和程度。

如果把方差看成“自己和自己的关系”，那么协方差就是“两个变量之间的关系”。

从公式上看：

Var(X) = E[(X - E(X))²]

协方差是：

Cov(X, Y) = E[(X - E(X))(Y - E(Y))]

所以方差其实可以看作协方差的特殊情况：

Var(X) = Cov(X, X)

18. 方差和均方误差

方差看的是数据围绕自身均值的波动：

Var(X) = E[(X - μ)²]

均方误差看的是预测值和真实值之间的误差：

MSE = E[(Y - Ŷ)²]

它们形式相似，都是平方后取平均。但含义不同：

  • 方差关注数据自身的离散程度。
  • MSE 关注模型预测的错误程度。

第八部分：从代码角度理解方差

19. Python 计算方差

如果手写计算总体方差：

data = [2, 4, 6, 8]

mean = sum(data) / len(data)
variance = sum((x - mean) ** 2 for x in data) / len(data)

print(mean)      # 5.0
print(variance)  # 5.0

计算样本方差：

data = [2, 4, 6, 8]

mean = sum(data) / len(data)
sample_variance = sum((x - mean) ** 2 for x in data) / (len(data) - 1)

print(sample_variance)  # 6.666666666666667

使用 NumPy：

import numpy as np

data = np.array([2, 4, 6, 8])

print(np.var(data))            # 总体方差，默认除以 n
print(np.var(data, ddof=1))    # 样本方差，除以 n - 1
print(np.std(data))            # 总体标准差
print(np.std(data, ddof=1))    # 样本标准差

ddof 表示 delta degrees of freedom，也就是自由度调整：

  • ddof=0：除以 n。
  • ddof=1：除以 n - 1。

第九部分：学习方差时最重要的几个结论

20. 核心总结

方差的本质是：

衡量数据围绕平均值的波动程度。

你可以按这个顺序理解它：

  1. 先计算平均值，找到数据中心。
  2. 再计算每个数据到中心的偏差。
  3. 为了避免正负抵消，把偏差平方。
  4. 对平方后的偏差取平均，得到方差。
  5. 如果想要更直观的单位，就对方差开方，得到标准差。

几个关键点：

  • 平均数回答“数据集中在哪里”。
  • 方差回答“数据分散得有多厉害”。
  • 方差越大，数据越不稳定。
  • 方差越小，数据越集中。
  • 总体方差除以 n。
  • 样本方差通常除以 n - 1。
  • 标准差是方差的平方根，更适合直观解释。
  • 方差对离群值敏感，因为偏差被平方了。

21. 最后用一句话记住方差

如果平均值是一组数据的“中心位置”，那么方差就是这组数据围绕中心的“散开程度”。

它让我们不再只问：

平均是多少？

而是进一步追问：

数据稳定吗？波动大吗？风险高吗？差异明显吗？

这就是方差真正重要的地方。
