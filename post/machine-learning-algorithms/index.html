机器学习算法全景图：一口气看完50种核心算法

2026-04-11 21:18 76 min read 人阅读

机器学习算法百科全书：分类、回归、聚类、强化一网打尽

引言：算法不是用来背的，是用来查的

初学者面对机器学习时，最头疼的事情莫过于——算法太多了。

线性回归、逻辑回归、决策树、随机森林、SVM、KNN、K-Means、DBSCAN、PCA、LDA、XGBoost、LightGBM、CatBoost、RNN、LSTM、GRU、CNN、Transformer、GAN、VAE、DQN、PPO……

光是看到这一串名字，很多人就已经想关掉页面了。

但请先别跑。这篇文章的目的，不是让你记住每一个算法的细节，而是给你一张地图。你只需要知道：

  • 这个算法属于哪一类？
  • 它主要解决什么问题？
  • 什么时候该考虑用它？

至于每个算法内部的数学原理和代码实现，后续我会逐一单独写文章深度拆解。今天，我们先把整个家族的族谱理清楚。

第一章：算法分类的总框架

在深入具体算法之前，先用一张思维导图建立全局视角：

机器学习算法
│
├── 按学习方式分
│   ├── 监督学习（有标签）
│   ├── 无监督学习（无标签）
│   ├── 半监督学习（少量标签）
│   ├── 自监督学习（从数据内部生成标签）
│   └── 强化学习（与环境交互，有奖励）
│
├── 按模型类型分
│   ├── 线性模型
│   ├── 树模型与集成学习
│   ├── 神经网络与深度学习
│   ├── 概率图模型
│   ├── 基于实例的模型
│   └── 降维与特征工程方法
│
└── 按任务类型分
    ├── 回归（预测连续值）
    ├── 分类（预测离散类别）
    ├── 聚类（无监督分组）
    ├── 降维（数据压缩）
    ├── 异常检测（找异常点）
    ├── 关联规则（找频繁模式）
    └── 序列决策（强化学习）

接下来的内容，将按照学习方式为主线，模型类型为辅线，把主流的、次主流的、经典的、前沿的算法全部梳理一遍。

第二章：监督学习算法——有答案的学习

监督学习是机器学习中最成熟、应用最广的领域。根据任务类型，分为回归和分类两大类，但很多算法可以同时处理两者。

2.1 线性模型家族

算法名称 一句话定位 核心特点
线性回归 回归任务的“Hello World” 假设目标与特征呈线性关系，用最小二乘法拟合一条直线
多项式回归 线性回归的曲线扩展版 将特征做多项式变换后再套线性回归，能拟合曲线
岭回归 带L2正则化的线性回归 防止过拟合，收缩系数但不归零
Lasso回归 带L1正则化的线性回归 能把不重要特征的系数压到零，自带特征选择
弹性网络 岭回归+Lasso的混合体 同时使用L1和L2正则化，取两者之长
逻辑回归 分类任务的“Hello World”（尽管名字叫回归） 在线性组合外套一层Sigmoid函数，输出类别概率
线性判别分析 有监督的降维+分类器 寻找能把不同类别最大程度分开的投影方向

2.2 基于距离/实例的模型

算法名称 一句话定位 核心特点
K近邻（KNN） 最懒惰的算法，几乎不训练 新样本的类别由最近的K个邻居投票决定
半径近邻 KNN的变体 以新样本为中心画一个半径，圈内所有邻居投票
最近质心分类器 比KNN更懒的分类器 每个类别只算一个质心，新样本归到最近的质心

2.3 支持向量机家族

算法名称 一句话定位 核心特点
线性SVM 找一条最宽的“隔离带”分开两类 最大化分类间隔，只关心边界上的支持向量
核SVM 用“核技巧”把非线性问题变线性 通过核函数将数据映射到高维空间再切分
SVR SVM的回归版本 找一个“管道”，让尽可能多的点落在管道内
One-Class SVM 单类分类，用于异常检测 只学“正常数据”的边界，边界外即为异常

2.4 决策树与集成学习家族

算法名称 一句话定位 核心特点
ID3 / C4.5 / CART 决策树的三个经典版本 通过不断选择最优特征进行分裂，构建树结构
随机森林 种一片树，投票决定结果 并行训练多棵决策树，每棵树看不同的样本和特征
AdaBoost 错题重做，越错越重视 串行训练弱分类器，每次加大上一轮分错样本的权重
GBDT（梯度提升决策树） 每一棵树都在拟合前一棵的残差 串行集成，用梯度下降思想优化损失函数
XGBoost GBDT的工业级加强版 二阶导数近似、正则化、并行化、缺失值处理
LightGBM 微软出品，速度狂魔 基于直方图的算法，单边梯度采样，Leaf-wise生长
CatBoost 俄罗斯搜索巨头出品 天然支持类别特征，无需手动编码，对过拟合鲁棒
NGBoost 概率预测版梯度提升 输出预测值的同时输出预测的不确定性
HistGradientBoosting sklearn内置的快速梯度提升 受LightGBM启发，sklearn原生实现，性能优异

2.5 朴素贝叶斯家族

算法名称 一句话定位 核心特点
高斯朴素贝叶斯 假设特征服从正态分布 适合连续型特征
多项式朴素贝叶斯 假设特征是离散计数 文本分类经典选择，适合词频特征
伯努利朴素贝叶斯 假设特征是二值变量 适合“出现/不出现”的二值特征
补集朴素贝叶斯 多项式贝叶斯的改进版 专门处理类别不平衡问题

2.6 其他经典监督算法

算法名称 一句话定位 核心特点
感知机 神经网络的单细胞祖先 线性二分类器，Minsky和Papert的经典模型
神经网络（多层感知机） 万能函数逼近器 由多个神经元层堆叠而成，深度学习的基础
高斯过程回归 带不确定性度量的回归 输出一个概率分布而非一个点估计
保序回归 保持顺序约束的回归 预测值必须保持与某个变量单调关系的约束
分位数回归 不只预测均值，可预测任意分位数 用于风险评估和区间预测
贝叶斯回归 把贝叶斯推断引入线性回归 对模型参数引入先验分布，输出后验分布

第三章：无监督学习算法——没有答案的探索

无监督学习不依赖标签，试图从数据本身发现隐藏的结构。

3.1 聚类算法家族

算法名称 一句话定位 核心特点
K-Means 最经典的划分式聚类 选K个中心，迭代分配样本到最近中心，更新中心
K-Medoids K-Means的鲁棒升级版 用实际样本点作为中心，对异常值更鲁棒
Mini-Batch K-Means 大数据版K-Means 每次用小批量样本更新中心，大幅加速
层次聚类 自底向上或自顶向下构建树状结构 不需要预设K值，结果可呈现为树状图
DBSCAN 基于密度的聚类，能识别任意形状 通过密度相连的点聚成簇，自动识别噪声
OPTICS DBSCAN的改进版 对密度不均匀的数据适应更好
HDBSCAN DBSCAN的进一步升级 自适应密度阈值，更稳定，更少参数
均值漂移 往密度高的地方“漂移” 通过迭代找到数据密度中心，自动确定簇数
高斯混合模型（GMM） 用多个高斯分布混合拟合数据 软聚类，输出样本属于每个簇的概率
谱聚类 在图拉普拉斯矩阵上做聚类 适合非凸形状的簇，但计算代价高
BIRCH 专为超大数据库设计的聚类 用CF Tree压缩数据再聚类
Affinity Propagation 样本之间互相“推举”代表点 不需要预设K值，由偏好参数控制

3.2 降维算法家族

算法名称 一句话定位 核心特点
PCA（主成分分析） 降维界的第一工具 寻找方差最大的投影方向，线性降维
核PCA PCA的非线性版本 用核技巧处理非线性结构
SVD（奇异值分解） 矩阵分解的基础工具 PCA的底层数学实现，也用于推荐系统
LDA（线性判别分析） 有监督降维 投影后同类靠近，异类远离
t-SNE 可视化神器 专门将高维数据映射到2D/3D，保持局部结构
UMAP t-SNE的强力竞争者 速度更快，保留更多全局结构
Isomap 测地线距离降维 保持数据点在流形上的测地距离
LLE（局部线性嵌入） 保持局部线性关系 假设每个点可由邻居线性重构
MDS（多维缩放） 保持样本间距离的降维 输入距离矩阵，输出低维坐标
因子分析 寻找隐变量 假设观测变量由少数潜在因子生成
独立成分分析（ICA） 分离混合信号 假设信号源相互独立，用于盲源分离
非负矩阵分解（NMF） 分解为非负矩阵乘积 常用于文本主题建模和推荐系统
自编码器 神经网络做非线性降维 通过重构瓶颈层学习压缩表示

3.3 异常检测算法家族

算法名称 一句话定位 核心特点
孤立森林 用树分离孤立点 异常点更容易被随机划分快速孤立
局部异常因子（LOF） 基于密度的局部异常检测 比较一个点与邻居的局部密度
单类SVM SVM做异常检测 学习正常数据的边界，边界外判为异常
椭圆包络 假设数据服从高斯分布 用马氏距离度量异常程度
COPOD 基于Copula的异常检测 无需调参，计算高效

3.4 关联规则与频繁模式挖掘

算法名称 一句话定位 核心特点
Apriori 关联规则挖掘鼻祖 利用“频繁项集的子集也频繁”剪枝
FP-Growth 比Apriori快得多 用FP树压缩数据，不生成候选集
Eclat 深度优先搜索频繁项集 使用垂直数据格式，交集操作快速

第四章：集成学习算法——三个臭皮匠，顶个诸葛亮

集成学习本身是一种策略而非单一算法，但因其极端重要，单独成章。

算法名称 一句话定位 核心特点
Bagging 并行训练同质弱学习器，投票/平均 随机森林是其代表
Boosting 串行训练，后一个纠正前一个的错误 AdaBoost、GBDT系列均属此类
Stacking 用另一个模型融合多个模型的输出 多层模型堆叠，第二层学习第一层的最佳组合
Blending Stacking的简化版 只用留出集训练元模型，防止过拟合
Voting 最朴素的集成：硬投票/软投票 多个模型投票决定最终结果

第五章：神经网络与深度学习算法——多层抽象的艺术

深度学习是机器学习的子领域，近年来几乎主导了CV、NLP和语音等非结构化数据任务。

5.1 基础网络架构

算法名称 一句话定位 核心特点
多层感知机 全连接神经网络 层与层之间所有神经元互相连接
卷积神经网络 图像处理的标配 局部连接、权值共享、池化下采样
循环神经网络 处理序列数据的基础 隐状态在时间步之间传递信息
LSTM 长短期记忆网络 引入门控机制解决RNN的梯度消失
GRU LSTM的简化高效版 合并遗忘门和输入门，参数更少

5.2 高级与专用架构

算法名称 一句话定位 核心特点
ResNet 残差网络，让深层网络可训练 跳跃连接，恒等映射，解决梯度消失
DenseNet 密集连接网络 每一层都连接到后面所有层
Inception / GoogLeNet 多尺度并行卷积 同一层用不同尺寸卷积核
MobileNet 轻量级移动端CNN 深度可分离卷积大幅减少参数
EfficientNet 用神经架构搜索找到的最优缩放 在深度、宽度、分辨率三者间平衡
U-Net 图像分割专用架构 对称编解码结构，跳跃连接保留细节
YOLO / SSD / Faster R-CNN 目标检测经典系列 端到端检测，速度与精度的不同取舍
Transformer 颠覆NLP乃至整个深度学习的架构 自注意力机制，并行处理序列
BERT 预训练语言模型的里程碑 双向Transformer编码器，MLM预训练
GPT系列 自回归语言模型 单向Transformer解码器，预测下一个词
ViT 把Transformer用于图像 将图像切块当作序列输入
GAN 生成对抗网络 生成器与判别器博弈，生成逼真样本
VAE 变分自编码器 学习隐空间概率分布，可控生成
扩散模型 当前最强的图像生成模型 逐步加噪再学习去噪，Stable Diffusion的基础
图神经网络 处理图结构数据的神经网络 节点信息沿图边传播更新

5.3 训练技巧与优化算法

算法名称 一句话定位
SGD 最基础的梯度下降优化器
Momentum 引入动量，加速SGD，减少震荡
Adam 自适应学习率，最常用的优化器
AdamW Adam + 权重衰减解耦，效果更好
RMSprop 适合非平稳目标和RNN
Batch Normalization 对每一层输入归一化，加速训练
Layer Normalization Transformer标配归一化方法
Dropout 随机丢弃神经元，防过拟合
Early Stopping 验证集性能不再提升时停止训练
Learning Rate Scheduling 动态调整学习率
Gradient Clipping 梯度截断，防止梯度爆炸

第六章：强化学习算法——在试错中成长

强化学习让智能体通过与环境交互学习最优策略。

算法名称 一句话定位 核心特点
Q-Learning 基于表格的价值学习鼻祖 学习状态-动作价值函数Q，离线策略
SARSA Q-Learning的在线策略版 行动策略与更新策略一致
DQN 深度Q网络 用神经网络拟合Q函数，引入经验回放和目标网络
Double DQN 解决DQN高估问题 用当前网络选动作，目标网络评估价值
Dueling DQN 分离状态价值和动作优势 网络输出分为V和A两部分再合并
Policy Gradient 直接优化策略函数 输出动作的概率分布而非价值
REINFORCE 最基础的策略梯度算法 蒙特卡洛采样计算梯度
Actor-Critic 价值网络+策略网络结合 Actor输出动作，Critic评估好坏
A2C / A3C 优势Actor-Critic，同步/异步 用优势函数代替原始回报减少方差
PPO 当前最流行的策略梯度算法 限制策略更新幅度，稳定训练
TRPO 信任域策略优化 PPO的前身，数学更严谨但实现更复杂
DDPG 深度确定性策略梯度 将DQN思想扩展到连续动作空间
TD3 DDPG改进版 双Q网络、延迟策略更新、目标策略平滑
SAC 软Actor-Critic 引入最大熵，鼓励探索，样本效率高
IMPALA 大规模分布式强化学习 分离学习器和执行器，高吞吐量
AlphaGo / AlphaZero 围棋AI的巅峰 蒙特卡洛树搜索+深度神经网络

第七章：概率图模型——用图表示变量依赖关系

这类算法在结构化预测、因果推断中扮演重要角色。

算法名称 一句话定位 核心特点
朴素贝叶斯 最简单的有向图模型 假设特征条件独立
隐马尔可夫模型（HMM） 序列标注经典模型 观测序列+隐藏状态链，状态转移满足马尔可夫性
最大熵马尔可夫模型（MEMM） HMM改进版 允许非独立特征，但有标注偏置问题
条件随机场（CRF） 序列标注的统计学习基石 全局归一化，解决标注偏置
贝叶斯网络 有向无环图表示变量依赖 可用于推理、预测、因果发现
马尔可夫随机场 无向图模型 图像分割、去噪常用

第八章：其他重要算法与概念补充

8.1 特征工程相关算法

算法名称 一句话定位
One-Hot Encoding 将类别特征转为二值向量
Label Encoding 将类别映射为整数
Target Encoding 用目标变量的统计量替换类别
特征哈希 将高基数特征映射到固定维度
分箱/离散化 将连续值分段转为类别
多项式特征生成 生成特征的交互项和高次项

第九章：算法选型速查指南

面对一个具体问题，如何快速缩小算法选择范围？

你的问题描述 优先考虑的算法方向
数据有标签，预测连续值 线性回归 → 岭回归/Lasso → XGBoost/LightGBM → 神经网络
数据有标签，预测类别 逻辑回归 → 随机森林 → XGBoost/LightGBM → 神经网络
数据有标签，但数据量巨大（百万级） LightGBM → 深度学习
数据无标签，想分组 K-Means → DBSCAN/HDBSCAN → 高斯混合模型
数据无标签，想压缩到2D/3D可视化 t-SNE → UMAP
数据无标签，想发现异常 孤立森林 → 局部异常因子
数据是图片 CNN（ResNet / EfficientNet） → ViT
数据是文本 TF-IDF+朴素贝叶斯/逻辑回归 → BERT/GPT
数据是时间序列 ARIMA/Prophet → LSTM/GRU → Transformer
数据是图/关系网络 图神经网络（GCN/GAT）
需要决策/控制（如下棋、自动驾驶） 强化学习（PPO / SAC / DQN）
