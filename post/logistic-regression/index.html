从线性回归到逻辑回归：深入理解分类模型的灵魂

2026-04-26 22:41 498 min read 人阅读

逻辑回归：机器学习中最被低估的“分类之王

引言：一个名字引发的世纪误会

如果你问一个机器学习初学者：“逻辑回归是用来做什么的？”得到的答案很可能是：“做回归的，预测数值。”

这个误会几乎成了机器学习入门的必经之路。逻辑回归（Logistic Regression）虽然名字里带着“回归”两个字，但它是最经典、最常用的二分类算法之一。

更令人惊讶的是，在工业界的大规模推荐系统和点击率预估任务中，逻辑回归至今仍然是强基线模型，其地位完全不逊于深度学习模型。原因在于：

  • 概率输出：不仅输出类别，还输出属于某个类别的概率。
  • 计算高效：训练和预测速度极快，可以处理海量数据。
  • 高可解释性：每个特征的权重都有明确的含义。
  • 易于正则化：可以应对高维稀疏特征。

本文将从头拆解逻辑回归的每一个细节，用尽量直觉化的方式讲清楚Sigmoid函数、交叉熵损失、梯度推导，并附上完整的可执行Python代码。

第一章：为什么线性回归不能做分类？

1.1 线性回归的硬伤

假设我们要做一个垃圾邮件分类器。我们给模型输入邮件的特征（如包含“免费”这个词的次数），希望它输出“是垃圾邮件”或“不是垃圾邮件”。

如果我们直接用线性回归：

y=wx+by = w x + b y=wx+b

会遇到三个致命问题：

问题一：输出范围不受限

线性回归的输出 yyy 可以是任何实数（−∞-\infty−∞ 到 +∞+\infty+∞）。但我们想要的是 0 或 1（或者一个介于0到1之间的概率）。用直线去拟合0/1标签，效果极差。

问题二：对异常值极度敏感

如果突然出现一个样本，特征 xxx 特别大，线性回归的拟合线会被这个异常点严重拉偏，导致整体分类效果崩塌。

问题三：误差衡量不合理

在线性回归中，我们使用均方误差（MSE）来衡量预测值与真实值的差距。但在分类任务中，预测值0.8和真实值1之间的“误差”，与预测值0.1和真实值0之间的“误差”，不应该被同等对待——我们更需要一个专门衡量“概率预测准确性”的损失函数。

结论：我们需要一个专门的分类模型。

第二章：逻辑回归的核心思想

2.1 从回归到分类：加一层“转换函数”

逻辑回归的解决方案极其优雅：在线性回归的输出上，套一层Sigmoid函数，将任意实数映射到 (0,1) 区间，这个值就可以解释为概率。

整个模型的数学表达式为：

线性组合：

z=w1x1+w2x2+⋯+wnxn+bz = w_1 x_1 + w_2 x_2 + \cdots + w_n x_n + b z=w1​x1​+w2​x2​+⋯+wn​xn​+b

概率映射：

p^=σ(z)=11+e−z\hat{p}=\sigma(z)=\frac{1}{1+e^{-z}} p^​=σ(z)=1+e−z1​

其中 σ(⋅)\sigma(\cdot)σ(⋅) 就是 Sigmoid 函数，它的图像是一条优美的S形曲线。

2.2 Sigmoid 函数的数学性质

Sigmoid 函数有几条极其重要的性质，这些性质让逻辑回归的数学推导变得异常简洁：

性质一：值域为 (0,1)

lim⁡z→−∞σ(z)=0,lim⁡z→+∞σ(z)=1\lim_{z\to -\infty}\sigma(z)=0,\quad \lim_{z\to +\infty}\sigma(z)=1 z→−∞lim​σ(z)=0,z→+∞lim​σ(z)=1

性质二：关于 (0, 0.5) 中心对称

σ(0)=0.5\sigma(0)=0.5 σ(0)=0.5

这为决策阈值的选取提供了自然的参考点：当 p^≥0.5\hat{p} \ge 0.5p^​≥0.5 时，预测为正类；否则预测为负类。

性质三：导数可以用自身表示（魔鬼性质）

σ′(z)=σ(z)(1−σ(z))\sigma'(z)=\sigma(z)\left(1-\sigma(z)\right) σ′(z)=σ(z)(1−σ(z))

这条性质会让你在后续梯度推导时大吃一惊——计算导数几乎零成本。

2.3 决策边界：逻辑回归的几何解释

当我们以 0.5 为阈值进行分类时：

  • σ(z)≥0.5  ⟺  z≥0\sigma(z) \ge 0.5 \quad \iff \quad z \ge 0σ(z)≥0.5⟺z≥0（因为 σ(0)=0.5\sigma(0)=0.5σ(0)=0.5 且单调递增）
  • σ(z)<0.5  ⟺  z<0\sigma(z) < 0.5 \quad \iff \quad z < 0σ(z)<0.5⟺z<0

因此，决策条件转化为：

z=wTx+b=0z=\mathbf{w}^{T}\mathbf{x}+b=0 z=wTx+b=0

这就是决策边界（Decision Boundary） 的方程。在特征空间中，它是一个由 wTx+b=0\mathbf{w}^T \mathbf{x} + b = 0wTx+b=0 定义的超平面。一侧被分类为正，另一侧为负。从这个角度看，逻辑回归本质上是一个线性分类器。

第三章：损失函数——逻辑回归的灵魂

3.1 为什么不用均方误差（MSE）？

如果我们将 Sigmoid 输出视为 p^\hat{p}p^​，真实标签为 y∈0,1y \in {0,1}y∈0,1，套用 MSE：

J=1n∑i=1n(p^i−yi)2J=\frac{1}{n}\sum_{i=1}^{n}\left(\hat{p}_i-y_i\right)^2 J=n1​i=1∑n​(p^​i​−yi​)2

这个函数关于参数 w\mathbf{w}w 和 bbb 是非凸的，存在多个局部最小值，梯度下降会陷入局部最优解。因此，我们需要一个凸的损失函数。

3.2 极大似然估计（MLE）与交叉熵损失

逻辑回归的损失函数来源于统计学中的极大似然估计（Maximum Likelihood Estimation）。

对于一个样本，如果真实标签 y=1y=1y=1，我们希望预测概率 p^\hat{p}p^​ 越接近1越好；如果 y=0y=0y=0，我们希望 p^\hat{p}p^​ 越接近0越好。我们可以用一条公式统一这两种情况：

L(单样本)=p^ y(1−p^)1−y\mathcal{L}(\text{单样本})=\hat{p}^{\,y}\left(1-\hat{p}\right)^{1-y} L(单样本)=p^​y(1−p^​)1−y

  • 当 y=1y=1y=1 时，上式退化为 p^\hat{p}p^​。
  • 当 y=0y=0y=0 时，上式退化为 1−p^1 - \hat{p}1−p^​。

为了累乘方便，并且把最大化问题转化为更熟悉的最小化问题，我们对全体样本取负对数似然：

J=−1n∑i=1n[yiln⁡p^i+(1−yi)ln⁡(1−p^i)]J=-\frac{1}{n}\sum_{i=1}^{n}\left[y_i\ln \hat{p}_i+(1-y_i)\ln\left(1-\hat{p}_i\right)\right] J=−n1​i=1∑n​[yi​lnp^​i​+(1−yi​)ln(1−p^​i​)]

这就是著名的二元交叉熵损失（Binary Cross-Entropy Loss），也叫对数损失。

3.3 交叉熵为什么好？

  • 凸性：在参数空间中，交叉熵损失是凸函数，保证了梯度下降能找到全局最小值。
  • 梯度性质好：当预测值与真实值相差很大时，梯度很大，更新步幅大，学习快；当预测接近真实值时，梯度变小，学习慢，收敛稳定。
  • 信息论解释：交叉熵衡量的是真实分布与预测分布之间的差异。

第四章：梯度推导与参数优化

4.1 单样本梯度的链式法则推导

我们用单样本的交叉熵损失函数来推导梯度。设 a=p^=σ(z)a = \hat{p} = \sigma(z)a=p^​=σ(z)，z=wTx+bz = \mathbf{w}^T \mathbf{x} + bz=wTx+b，损失为：

L=−[yln⁡a+(1−y)ln⁡(1−a)]L=-\left[y\ln a+(1-y)\ln(1-a)\right] L=−[ylna+(1−y)ln(1−a)]

我们要计算 ∂L∂wj\frac{\partial L}{\partial w_j}∂wj​∂L​ 和 ∂L∂b\frac{\partial L}{\partial b}∂b∂L​。

第一步：损失对 aaa 的导数

∂L∂a=−(ya−1−y1−a)=a−ya(1−a)\frac{\partial L}{\partial a}=-\left(\frac{y}{a}-\frac{1-y}{1-a}\right)=\frac{a-y}{a(1-a)} ∂a∂L​=−(ay​−1−a1−y​)=a(1−a)a−y​

第二步：aaa 对 zzz 的导数（利用 Sigmoid 的魔鬼性质）

∂a∂z=a(1−a)\frac{\partial a}{\partial z}=a(1-a) ∂z∂a​=a(1−a)

第三步：zzz 对参数 wjw_jwj​ 和 bbb 的导数

∂z∂wj=xj,∂z∂b=1\frac{\partial z}{\partial w_j}=x_j,\quad \frac{\partial z}{\partial b}=1 ∂wj​∂z​=xj​,∂b∂z​=1

由链式法则，合并后奇迹发生：

∂L∂wj=(a−y)xj,∂L∂b=a−y\frac{\partial L}{\partial w_j}=(a-y)x_j,\quad \frac{\partial L}{\partial b}=a-y ∂wj​∂L​=(a−y)xj​,∂b∂L​=a−y

4.2 完整梯度（n个样本）

对于 nnn 个样本，总损失对参数的梯度为：

∂J∂wj=1n∑i=1n(ai−yi)xij,∂J∂b=1n∑i=1n(ai−yi)\frac{\partial J}{\partial w_j}=\frac{1}{n}\sum_{i=1}^{n}(a_i-y_i)x_{ij},\quad \frac{\partial J}{\partial b}=\frac{1}{n}\sum_{i=1}^{n}(a_i-y_i) ∂wj​∂J​=n1​i=1∑n​(ai​−yi​)xij​,∂b∂J​=n1​i=1∑n​(ai​−yi​)

这个梯度的形式极其简洁优雅：梯度等于预测值与真实值之差乘以特征值，这是前面 Sigmoid 函数导数巧妙抵消后的结果。

4.3 参数更新公式

使用梯度下降法，参数更新规则为：

wj←wj−α1n∑i=1n(ai−yi)xij,b←b−α1n∑i=1n(ai−yi)w_j \leftarrow w_j-\alpha \frac{1}{n}\sum_{i=1}^{n}(a_i-y_i)x_{ij},\quad b \leftarrow b-\alpha \frac{1}{n}\sum_{i=1}^{n}(a_i-y_i) wj​←wj​−αn1​i=1∑n​(ai​−yi​)xij​,b←b−αn1​i=1∑n​(ai​−yi​)

其中 α\alphaα 为学习率。

第五章：正则化——高维稀疏场景的利器

在实际应用中，逻辑回归经常面对特征数远多于样本数的高维稀疏场景（如文本分类中的TF-IDF特征），极容易过拟合。正则化在此至关重要。

5.1 L2 正则化（岭惩罚）

在损失函数后添加权重平方和的惩罚项：

JL2=Jorigin+λ2n∑j=1mwj2J_{L2}=J_{\text{origin}}+\frac{\lambda}{2n}\sum_{j=1}^{m}w_j^2 JL2​=Jorigin​+2nλ​j=1∑m​wj2​

梯度变为：

∂JL2∂wj=1n∑i=1n(ai−yi)xij+λnwj\frac{\partial J_{L2}}{\partial w_j}=\frac{1}{n}\sum_{i=1}^{n}(a_i-y_i)x_{ij}+\frac{\lambda}{n}w_j ∂wj​∂JL2​​=n1​i=1∑n​(ai​−yi​)xij​+nλ​wj​

  • 效果：将权重向零压缩，但所有特征都有非零系数。
  • 适用场景：特征间存在共线性，或者担心丢掉某些有用信息。

5.2 L1 正则化（Lasso惩罚）

添加权重绝对值之和的惩罚项：

JL1=Jorigin+λn∑j=1m∣wj∣J_{L1}=J_{\text{origin}}+\frac{\lambda}{n}\sum_{j=1}^{m}\left|w_j\right| JL1​=Jorigin​+nλ​j=1∑m​∣wj​∣

  • 效果：产生稀疏解，大量不重要特征的权重被直接压缩到0。
  • 适用场景：特征数量极大，需要自动特征选择。

5.3 弹性网络

结合L1和L2：

JElnet=Jorigin+λ1n∑j=1m∣wj∣+λ22n∑j=1mwj2J_{\text{Elnet}}=J_{\text{origin}}+\frac{\lambda_1}{n}\sum_{j=1}^{m}\left|w_j\right|+\frac{\lambda_2}{2n}\sum_{j=1}^{m}w_j^2 JElnet​=Jorigin​+nλ1​​j=1∑m​∣wj​∣+2nλ2​​j=1∑m​wj2​

  • 适用场景：特征数远大于样本数，且需要保留群体效应。

第六章：从二分类到多分类——Softmax回归

逻辑回归只能处理二分类。如果要做多分类（例如识别手写数字0-9），需要推广。Softmax回归 为每个类别都学习一套权重向，然后通过Softmax函数将评分转化为概率分布。

对于 KKK 个类别，每个类别有一个得分：

zk=wkTx+bkz_k=\mathbf{w}_k^{T}\mathbf{x}+b_k zk​=wkT​x+bk​

Softmax 函数将得分转化为概率：

p^k=ezk∑j=1Kezj\hat{p}_k=\frac{e^{z_k}}{\sum_{j=1}^{K}e^{z_j}} p^​k​=∑j=1K​ezj​ezk​​

预测时选择概率最大的类别作为输出。损失函数为交叉熵的推广：

J=−1n∑i=1n∑k=1Kyikln⁡p^ikJ=-\frac{1}{n}\sum_{i=1}^{n}\sum_{k=1}^{K}y_{ik}\ln \hat{p}_{ik} J=−n1​i=1∑n​k=1∑K​yik​lnp^​ik​

其中 yiky_{ik}yik​ 是 one-hot 编码后的目标（若样本 iii 属于类别 kkk 则为1，否则0）。反向传播时梯度形式同样简洁——类似于二元逻辑回归的扩展。

第八章：深入核心概念——几率与对数几率

8.1 几率（Odds）

设 p^\hat{p}p^​ 为事件发生的概率，几率定义为：

Odds=p^1−p^\text{Odds}=\frac{\hat{p}}{1-\hat{p}} Odds=1−p^​p^​​

如果 p^=0.75\hat{p} = 0.75p^​=0.75，几率为3，意味着事件发生的可能性是不发生的3倍。

8.2 对数几率（Log-Odds / Logit）

对几率取自然对数，就得到了对数几率，也叫 Logit 函数：

Logit(p^)=ln⁡(p^1−p^)\text{Logit}(\hat{p})=\ln\left(\frac{\hat{p}}{1-\hat{p}}\right) Logit(p^​)=ln(1−p^​p^​​)

现在，我们看逻辑回归的决策核心：p^=σ(z),z=wTx+b\hat{p} = \sigma(z), z = \mathbf{w}^T\mathbf{x} + bp^​=σ(z),z=wTx+b。反过来：

p^1−p^=ez,ln⁡(p^1−p^)=z=wTx+b\frac{\hat{p}}{1-\hat{p}}=e^{z},\quad \ln\left(\frac{\hat{p}}{1-\hat{p}}\right)=z=\mathbf{w}^{T}\mathbf{x}+b 1−p^​p^​​=ez,ln(1−p^​p^​​)=z=wTx+b

这揭示了逻辑回归名字的由来：模型将对数几率建模为输入特征的线性组合。这就是“回归”的含义——它在线性地回归对数几率。因此，特征权重 wjw_jwj​ 的解释为：在其他特征不变的情况下，xjx_jxj​ 每增加一个单位，结果的对数几率增加 wjw_jwj​，或者说几率乘以 ewje^{w_j}ewj​。

第九章：类别不平衡问题与应对策略

在现实任务中（如欺诈检测、疾病筛查），正负样本比例可能极不平衡，此时简单以0.5为阈值会失效。

策略一：调整决策阈值

不改变模型本身，而是根据验证集上的PR曲线选择一个更合适的阈值，使F1-score或业务指标最优。

策略二：类别权重（Class Weight）

在损失函数中对少数类样本施加更大的惩罚权重。在Scikit-learn中，设置 class_weight='balanced' 即可自动调整权重。

策略三：重采样

  • 过采样少数类（如SMOTE）
  • 欠采样多数类

第十章：逻辑回归与现代深度学习的关系

逻辑回归是神经网络的基本单元。如果我们把多个逻辑回归单元堆叠起来，并使用非线性的激活函数（如ReLU），就构成了多层感知机。事实上，对于二分类问题，一个单神经元的神经网络，加上Sigmoid激活函数和二值交叉熵损失，就是逻辑回归的神经网络版本。

总结

逻辑回归作为机器学习的基石算法，用一条优美的Sigmoid曲线连接了线性回归和分类任务。它的推导过程展示了损失函数设计、极大似然估计、凸优化和正则化的精妙结合。
